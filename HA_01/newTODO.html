New Todo

Please enter a description of the new task. Be as concise as you can. Make sure the TODO does not already exist.



When needs your task to be finished?



Please define milestones and how much of the task they make up. A milestone is another todo. If it doesn't exist it will be created. If you don't define milestones (you don't have to) the progress of the task will be 0% until the task is marked as finished (see How to edit Todo).
