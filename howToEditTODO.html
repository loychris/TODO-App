How to edit a TODO

  1. Click the EDIT button of the TODO you want to edit.
  2. Now you can edit the TODO text and the deadline, update the progress, add a subtask, or mark a subtask as finished.
  3. If you add a subtask, please adjust the percentages of the existing subtasks so they add up to 100%.
  4. Finally, klick the SUBMIT button.
  5. You should now see your changes on the main page. If you have added a subtask, the progress bar should or should not have changed, depending on the percentage adjustments you have made.
